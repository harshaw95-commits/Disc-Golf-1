Camera-assisted disc profiling

Disc Flight Lab

Use your camera, a known-size reference object, and quick image annotations to estimate rim diameter, dome shape, and parting line height so you can better predict stability and flight.

How it works

  1. Start the rear camera and select the live measuring tool for either the top or side shot.
  2. Drag the calipers over the same reference and disc edges you want seeded into the captured measurement.
  3. Capture a top view to measure diameter and rim width.
  4. Capture a side profile to measure dome height and parting line height.
  5. Review the calculated profile and a heuristic stability summary.
